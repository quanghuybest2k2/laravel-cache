# redis cache

## Download

`https://github.com/MicrosoftArchive/redis/releases`

## command

```shell
redis-cli
```

### View all key

```shell
KEYS *

```

### View info key

```shell
GET <key>

```

### Delete key

```shell
DEL <key>

```

### View TTL

```shell
TTL <key>

```
